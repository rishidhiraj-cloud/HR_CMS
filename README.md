# HR Widget

Cross-platform desktop widget (Windows + macOS) that displays HR messages pushed via a web CMS panel.

Built with: Electron + React + Supabase + Next.js

## Structure

- `cms-panel/` — HR web panel (Next.js)
- `widget/` — Employee desktop widget (Electron)
- `supabase/` — Database migrations
- `docs/` — Design specs and implementation plans
